⚡ The Flywheel Effect

Low Context Switch Cost → More Energy IN Than OUT → Themes Lift Off

When switching terminals is cheap, you add more energy to the flywheel than you drain. Your tab bunches + AI in that terminal = compound momentum. Cmd+Space to the terminal name is instant context loading. After a week of repetition, switching to Kitty is not a context switch—it's context loading.

📋 Setup Instructions

1

Install ThetaCog (You Already Did This)

npm install -g thetacog-mcp
This page opened automatically. Your dashboards are now at ~/.thetacog/

2

Install Your Terminals

Pick 2-3 terminals from the list above. Don't have them? Ask Claude:
"Install Kitty and iTerm2 on my Mac"
Or install manually: Kitty · iTerm2 · WezTerm · Cursor

3

Bunch Your Browser Tabs by Theme

Group your tabs by what mode they serve:
• Revenue tabs: Stripe, CRM, analytics → keep near Kitty
• Code tabs: GitHub, CI/CD, API docs → keep near iTerm2
• Research tabs: Stack Overflow, papers → keep near WezTerm

4

Keep These Dashboards as Tabs

Open the room dashboards above and keep them as browser tabs:
• Operator dashboard tab → next to your Stripe tab
• Builder dashboard tab → next to your GitHub tab
Tip: Bookmark this folder: ~/.thetacog/

5

Split Screen: Terminal Left, Browser Right

macOS: Hover over the green maximize button → "Tile Window to Left" → Select browser for right side
Windows: Drag terminal to left edge until it snaps → Drag browser to right
Linux: Super+Left for terminal, Super+Right for browser

The Layout (always this pattern):
┌─────────────────┬─────────────────┐
│ LEFT │ RIGHT │
│─────────────────│─────────────────│
│ TERMINAL │ BROWSER TABS │

│ Kitty terminal │ Stripe.com │
│ $ your-ai │ CRM dashboard │
│ │ Analytics │
│ │ This dashboard │
└─────────────────┴─────────────────┘
6

Run Your AI in Each Terminal

Each terminal gets its own AI session for that stack's work. Any MCP-compatible client works (Claude Code, Cursor, Cline):
Cmd+Space "kitty" → AI for ops/business
Cmd+Space "iterm" → AI for building/coding
The jump is the insight. Instant context load.

7

Let Repetition Wire the Circuits

Hebbian learning: neurons that fire together, wire together. After a week, Cmd+Space "kitty" is instant context loading — not context switching. The jump is the insight.
